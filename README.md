# loan-calculator-frontend

Stack: Vue 3 and Vite.

## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Compile and Minify for Production

```sh
npm run build
```

### Lint with ESLint

```sh
npm run lint
```

### improvements

- tests on the frontend side
- custom validation instead of relying on HTML validation
